Overflow Scroll with Short-Message

This is a very long message that doesn't fit in the box. It forces both horizontal and vertical scrollbars to appear. You can scroll through the message to read it fully. The message is much wider than the container, which triggers horizontal scrolling. Additionally, the message may exceed the height of the box, triggering vertical scrolling. Scrollbars allow you to view all the text content, which keeps going and going without wrapping to the next line, so you have to scroll horizontally as well as vertically!.
